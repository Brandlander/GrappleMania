# GrappleMania
 This game is a 3rd person platformer where the player uses a grapple gun to scale the levels

# Controls
   WASD - Walking directions
   Left Shift - Run
   Space - Jump
   Right Mouse Button - Aim
   Left Mouse Button - Shoot
